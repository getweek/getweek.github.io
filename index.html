Week

Task management tool
with a heavy focus on planning

Features

My Day

Place where all your daily tasks live. See your planned tasks and upcoming events on a single page. All tasks due today are located here, but you can manually send tasks to this page to always keep an eye on them.

Calendar

Add new tasks to the calendar or drag and drop existing ones to plan them. Calendar is an important part of Week, so it is always within a reach.

Projects and Tags

There are multiple ways to group tasks in Week. You can create projects, tags or use subtasks - combine all these methods to unleash the full potential of the app.

Notes

Some tasks can't be done in few minutes and you might need to save your progress. Use notes with rich formatting, links and subtasks to write down important details and continue your work later.
